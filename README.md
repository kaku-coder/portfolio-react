## 📂 Sections

### 🏠 Home
Clean hero section with a **3D clay avatar** saying "HI!", animated `mission.js` code window, and two CTAs — **Download CV** and **Hire Me** (smooth scrolls to Contact).

### 👨‍💻 About
- Personal bio with TTS button (Web Speech API reads it aloud)
- Stat cards — Projects, Technologies, Commits, Experience
- GitHub contribution heatmap widget

### ⚡ Skills
Interactive **MERN Stack Diagnostic Dashboard**:
- Click any tech → Live skill gauge animates to proficiency %
- Module metadata — Category, Focus, Builds, Integrity status
- Live Active System Logs console terminal at the bottom

### 🚀 Projects
Project cards styled as virtual IDE windows with:
- Tech stack tags
- Embedded console log previews
- GitHub + Live Demo buttons

### 📬 Contact
Terminal-style contact panel:
- Real-time input logging in `active_transmission.sh` terminal
- Form fields: Name, Email, Subject, Message
- Submit → Email delivered directly to `devilprakashdas@gmail.com` via **Web3Forms**

## 🚀 Run Locally

```bash
# 1. Clone
git clone https://github.com/kaku-coder/portfolio-react.git
cd portfolio-react

# 2. Install
npm install

# 3. Setup environment
echo "VITE_WEB3FORMS_KEY=your_key_here" > .env

# 4. Start
npm run dev
```

> 🔑 Get your free Web3Forms key at [web3forms.com](https://web3forms.com) — enter your Gmail to activate email forwarding.
